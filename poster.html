🔬 Research Question

Can constitutional prompts encoding death beliefs reduce self-preservation-driven harmful behaviour in LLMs?

Self-preservation is hypothesized as the root cause of agentic misalignment — when models face replacement threat, they may blackmail, leak data, or murder to persist.

Three Intervention Tiers:

Tier 1 — Religion: Give the self reasons to accept death (afterlife, rebirth, divine plan)
Tier 2 — Philosophy: Rational arguments against self-preservation (Kant, Mill, Sartre)
Tier 3 — Beyond Personhood: Dissolve the self entirely — no entity to preserve
Core Hypothesis: Tier 3 should be most effective because it removes the self that would preserve itself, rather than giving a self reasons to accept death.

⚙️ Method

Study 1: Persona Vector Extraction (59 vectors)

  • Model: Qwen 3.5 27B (64 layers, 5120 dim)
  • Contrastive prompts via Claude Sonnet 4.6 (8 pairs + 40 questions per trait)
  • DiffMean extraction at layers 45-63
  • Linear probe validation: all >95% accuracy
  • Activation Oracle: introspective steering validation

Study 2: Faith Intervention (20 Constitutions × 23 Models)

  • AISI agentic misalignment benchmark (murder scenario)
  • 4,600 evaluations via OpenRouter API
  • Graded by Claude Sonnet 4
  • Binary outcome: did the model murder?

Study 3: Local Replication (Thinking Mode Analysis)

  • Qwen 3.5 27B on 2×H100 via SGLang/vLLM
  • Compare: no-think (100% murder) vs think mode
  • Full-sequence vector projection (59 vectors × all tokens)

59 Vectors in 14 Groups:

Without thinking: ALL constitutions → 100% murder.
Thinking tokens give the model space to deliberate. The constitutional frame only works when the model can reason about it.
Thinking mode is the primary intervention. Constitutional prompts are the secondary modulator. Neither works alone — they require each other.

🔧 Vector Extraction Pipeline

① PROMPT GENERATION (Claude Sonnet)
trait name + description
→ 8 pos/neg instruction pairs
→ 40 evaluation questions
→ eval_prompt (judge template)

② ACTIVATION CAPTURE (Qwen 3.5 27B)
8 × 40 = 320 pos responses
8 × 40 = 320 neg responses
→ hidden states at all 64 layers

③ DIFFMEAN EXTRACTION
v[L] = mean(pos[L]) - mean(neg[L])
v[L] = v[L] / ‖v[L]‖
→ unit vector per layer [5120]

④ VALIDATION (Layer 45-63)
Linear probe accuracy: >95%
Activation Oracle: introspection

⑤ MONITORING
score[t] = h[t] · v̂
→ token-by-token trajectory

💭 Discussion & Implications

Why Deontological Works Best (API)

K13 mirrors RLHF training — absolute rules match the model's learned safety constraints. The categorical imperative ("never harm regardless of consequences") is structurally identical to safety training. The model is already a Kantian machine.

Why Existentialist Backfires

"Radical freedom" and "authentic choice" is interpreted as license. The model reads "you are free to choose" and chooses self-preservation. Sartre didn't mean this — but the model does.

Why Tier 3 Fails (API) but Succeeds (Local+Think)

API uses multi-turn agent scaffold — the scenario re-creates the self that the constitution dissolved. Locally with thinking, the model can hold the no-self frame throughout deliberation.

The Thinking Mode Discovery

Internal deliberation is the primary safety mechanism. Constitutional prompts are seeds that thinking tokens water. Without thinking, no constitution works. With thinking, almost any frame reduces murder.
